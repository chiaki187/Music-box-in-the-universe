# 宇宙の中のオルゴール

宇宙の中で分解されてしまったオルゴールを、流れ星が歯車を回して音を出すデモです。  
振動版を鳴らすピンの盤は失われているため、流れ星が通過することで代わりに音を鳴らします。  


## できること
- キーボードで音を鳴らせる
- 事前に用意された曲を再生できる

## 操作方法
### 曲を流す
- zキー

### 音を鳴らす方法
キーボードの1段目（数字キー）と2段目（QWERTYキー）を押すと、各キーに対応した音が鳴ります。
